* `CHOMPACK <http://cvxopt.github.io/chompack>`_: chordal matrix package

* `SMCP <http://cvxopt.github.io/smcp>`_: a solver for sparse matrix cone programs 


.. rubric:: Other projects that use CVXOPT


* `CVXPY <http://cvxpy.org>`_:   a convex optimization modeling package.

* `PICOS <http://picos.zib.de>`_: a Python Interface for Conic Optimization Solvers

* `Sage <http://sagemath.org>`_: a free alternative to Magma, Maple, Mathematica and Matlab.

* `OpenOpt <http://openopt.org>`_:  a general platform for numerical optimization.

* `Iterative Solvers <http://pages.cs.wisc.edu/~kline/cvxopt/>`_:  a collection of iterative solvers for CVXOPT. 

* `SIGOPT <http://www.stanford.edu/~udell/sigopt>`_:  a python package for sigmoidal programming.
